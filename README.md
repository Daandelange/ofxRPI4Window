# ofxRPI4Window

### STATUS
Very early, no Mouse/Keyboard support

### DESCRIPTION:   
This is an openFrameworks addon for the Raspberry Pi to allow rendering without X

### REQUIREMENTS:   
- openFrameworks 11
- Raspberry Pi 4
- Raspbian Buster
- KMS Driver enabled

### USAGE:   
Clone into openFrameworks/addons
